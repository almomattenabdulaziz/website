web

SaudiChalets

built SaudiChalets, a full-stack chalet booking platform for the saudi market (basically a localized vacation rental / chalet reservation system) — for guest bookings, owner property management, and admin/moderation workflows.

built it with a production setup using Next.js (Pages Router), React, Node.js, and MariaDB (mysql2) with raw SQL + schema-adaptive query handling. it supports Arabic/English (rtl/ltr), responsive ui, multi-role permissions, booking lifecycle management, notifications, and ops tooling for real-world use.

key responsibilities & features

• multi-role platform (guest / owner / admin/moderator) with rbac permissions + secure access control
• property listings + search/filter (city, rating/reviews, facilities) + responsive layouts (mobile + desktop)
• e-mail authentication using SendGrid
• Whatsapp notifications and verification using Meta's WhatsApp API
• advanced pricing modes:
• fixed package pricing
• hourly pricing
• hourly + package overrides
• default/fallback pricing strategies based on booking rules
• booking flows:
• client booking creation
• owner confirm/decline workflows
• status lifecycle (pending / confirmed / cancelled)
• transition-safe handling for legacy/new status columns
• owner portal:
• property create/edit
• image uploads
• booking settings
• package pricing management
• dashboard + calendar views
• booking detail pages
• admin panel for ops (users/properties/bookings/reviews/support) + moderation (user disabling, hidden properties)
• support tickets (user + admin threads) with states, replies, and workflows
• email verification + password reset via SendGrid (secure tokens + expiry)
• pdf booking/invoice generation using Puppeteer + html templates (with arabic font embedding + booking snapshot data)
• audit logging foundation for key actions (auth, property changes, booking settings updates, etc.) + admin log viewer support
• localized ui polish across major pages (listings, booking, checkout, owner/admin pages)

technical highlights

• frontend: Next.js (Pages Router), React 18, css (scoped styles + responsive layout work)
• backend: Node.js api routes, MariaDB, raw sql (mysql2/promise), parameterized queries, transaction-safe patterns
• auth/security: session-based auth, role/permission checks, route/api protection, schema-adaptive guards, disabled-user enforcement
• files/media: multipart uploads (Formidable), property image storage/management
• docs/output: Puppeteer-generated pdfs with arabic font support + snapshot-based booking records
• integrations: SendGrid, payment workflow groundwork (Paymob KSA), Google Maps embeds/links
• localization: Arabic/English i18n, locale-aware routing, rtl/ltr handling
